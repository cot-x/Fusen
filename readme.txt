<<説明>>
    付箋プログラムです。
    起動すると、タスクトレイにアイコンが表示されます。
    右クリックで、操作が出来ます。
    また、各付箋も、右クリックによる操作が出来ます。
    各付箋は、ドラッグで、移動が出来ます。
    メモ帳代わりにでもどうぞ。
    付箋のデータと、ポジションは、保存されており、次回起動時に復元されます。
    スタートアップに登録し、常時起動状態にしておくと、便利かもしれません。
    
    このプログラムのリバースエンジニアリング、
        無許可な紹介、再配布、またはそれに類する事はできません。
    また、このプログラムを使用し、
        いかなる被害を被ったとしても、その責任は一切負いません。

<<動作環境>>
    .NET Framework 3.5以上の入った環境。
